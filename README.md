# FeeTrack Frontend - React + Vite

Modern mobile-first frontend for FeeTrack Library Management System.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Configuration

Edit `.env` file to set your backend API URL:
```
VITE_API_URL=/api
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production  
- `npm run preview` - Preview production build

## Features

✅ React 18 + Vite
✅ Tailwind CSS
✅ React Router 6
✅ JWT Authentication
✅ Mobile-first responsive design
✅ PWA support
✅ Role-based access (Admin/Library Owner)

## Tech Stack

- React 18 - UI library
- Vite - Build tool
- Tailwind CSS - Styling
- Axios - HTTP client
- React Router - Navigation
- React Hook Form - Forms
- Lucide React - Icons
